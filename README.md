# Soda Elementor Addons

A collection of premium custom Elementor widgets for creating stunning galleries, animated menus, and interactive elements.

## Version 2.3.0

A comprehensive plugin suite with multiple advanced widgets designed to enhance your Elementor page building experience.

## Features

### Menu Toggle Widget V2
- Animated hamburger menu toggle
- Customizable open/close text with 3D flip animations
- WordPress menu integration
- Multiple background scale animations (top, corner, right)
- Backdrop blur effects for modern glassmorphism
- Submenu support with smooth animations

### Gallery Widgets
- **Horizontal Gallery**: Smooth horizontal scrolling gallery with GSAP animations
- **Moving Gallery**: Dynamic gallery with motion effects
- **Pinned Gallery**: Scroll-triggered pinned image gallery
- **Zoom Gallery**: Interactive zoom effects on scroll
- **CubePortfolio**: Advanced portfolio grid with filtering

### Additional Widgets
- **Lottie Widget**: Animated Lottie JSON file player

### Advanced Modules
- **Parallax Background**: Smooth parallax effects for sections and containers
- **Liquid Glass**: Stunning liquid distortion effects with 10 presets (applies to any element)

### General Features
- Full Elementor customization controls
- Responsive design
- Box shadows and borders
- Custom icon library included
- GSAP-powered animations

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. All widgets will appear in Elementor's widget panel under "Soda Addons"

## Requirements

- WordPress 5.0 or higher
- Elementor 3.0.0 or higher
- PHP 7.0 or higher

## Usage

### Menu Toggle V2
1. Edit a page with Elementor
2. Search for "Menu Toggle V2" in the widget panel
3. Drag and drop it to your desired location
4. Select a WordPress menu from the dropdown
5. Customize colors, typography, animations, and effects

### Gallery Widgets
1. Add any gallery widget (Horizontal, Moving, Pinned, or Zoom Gallery)
2. Configure gallery settings and add your images
3. Customize animations, spacing, and visual effects
4. Preview and adjust responsive settings

### Lottie Widget
1. Add the Lottie Widget to your page
2. Upload or link to a Lottie JSON animation file
3. Configure playback settings and controls

### Liquid Glass Module
1. Select any Section, Container, Column, or Widget
2. Go to the Advanced tab
3. Find "Soda Liquid Glass" section
4. Enable the effect and choose from 10 presets
5. Tip: Use semi-transparent backgrounds for best results
